# Mivix UI Free Core And Pro Plan

Mivix UI uses a free core plus paid Pro model.

## Free Core

The `mivix-ui` package is the MIT-licensed core. It should remain useful enough for real products, prototypes, internal tools, dashboards, and learning.

Core includes:

- foundational UI components;
- theme tokens and light/dark/theme support;
- SSR and JSON rendering helpers;
- framework examples;
- basic charts and data display;
- docs, demos, and copy-pasteable examples.

## Future Pro Layer

The future Pro layer may be published separately, for example as `@mivix-ui/pro`, under a commercial license.

Good Pro candidates:

- advanced data table features;
- advanced chart interactions and analytics packs;
- full app templates for admin, SaaS, CRM, ecommerce, analytics, and AI workspaces;
- premium enterprise themes;
- rich text editor AI workflows;
- visual JSON form builder;
- Figma/design assets;
- priority fixes and private support;
- custom component development.

## Boundary Rule

The free core should earn trust. The Pro layer should save teams serious time.

Put a feature in Core when it is foundational, broadly useful, and important for adoption. Put it in Pro when it is advanced, enterprise-focused, support-heavy, or valuable as a ready-made business solution.

## Revenue Paths

Mivix can earn through:

- paid Pro npm package access;
- commercial template packs;
- enterprise support subscriptions;
- custom implementation services;
- accessibility and design-system audits;
- hosted builders or theme generators later.

## Licensing

The `mivix-ui` core package remains MIT licensed. Pro packages, template packs, and commercial services can use separate commercial terms.
